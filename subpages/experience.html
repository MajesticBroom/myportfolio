Technical subjects

Last June, we passed our first exam called "INF.02".
This exam included knowledge about computer networks,
network devices and server operating systems,
like Windows or Linux Server.

This year, we're going to pass second exam.
Passing both exams allows us to get Technical Diploma of IT Specialist.
New exam is called INF.03 and includes knowledge of web programming,
e.g. HTML, CSS, JavaScript, PHP or MySql.
